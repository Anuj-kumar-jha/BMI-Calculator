# 🏋️‍♂️ BMI Calculator

A simple **Body Mass Index (BMI) Calculator** built using **React.js**.

## 🚀 Features
- Calculate BMI based on **weight** and **height**.
- Displays BMI **category** (Underweight, Normal, Overweight, Obese).
- Simple and user-friendly **UI**.
